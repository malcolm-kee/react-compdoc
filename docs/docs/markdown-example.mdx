## Code Blocks

Different from the component examples, code blocks are static by default.

```js
const { buildShowroom } = require('react-showroom');

buildShowroom();
```

Make an example interative using `live` props.

```tsx live
const LiveExample = () => {
  const [count, setCount] = React.useState(0);

  return (
    <div className="p-4 text-center">
      <button
        type="button"
        onClick={() => setCount(count + 1)}
        className="px-6 py-2 w-24 bg-pink-700 text-white shadow rounded"
      >
        {count}
      </button>
    </div>
  );
};

<LiveExample />;
```

Code in `ts/js` can be run as well.

```ts live
setTimeout(() => {
  console.log('Hello World!');
}, 500);
```
